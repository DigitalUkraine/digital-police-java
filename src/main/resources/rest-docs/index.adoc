[[вступ]]
== Вступ

Ця документація являється посібником і описує методи взаємодії з REST API проекту Цифрова Поліція.
Подальші розділи описують окремі REST сервіси, їхнє призначення, приклади HTTP запитів та відповідей.

=== Мапа злочинів
